# Study-Buddy-extension

Want to focus on work but can't hold back checking  Instagram? well, here's a solution! 
<br><br>
Just add this extension to your browser and whenever you open any social media site, it will just replace the entire site content to a below mentioned 404 page, asking to get back to work :)<br><br>
